Detailed Description

template<typename CompletionToken, typename Signature>
struct asioext::async_completion< CompletionToken, Signature >

Helper template to deduce the handler type from a CompletionToken, capture a local copy of the handler, and then create an async_result for the handler.

Note
This type is the same as asio::async_completion if Asio 1.11.0+ is used. For prior versions this is a compatibility type that uses async_result to obtain the needed types.
