31 /// @brief Get a handle to the standard input.
32 ///
33 /// This function returns a handle to the standard input.
34 ///
35 /// @return A handle to the standard input (or an empty handle in case
36 /// of failure). Ownership is *not* transferred to the caller.
37 /// Closing the handle returned by this function leads to undefined behavior.
